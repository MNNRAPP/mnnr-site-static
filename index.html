How MNNR works

1) Define
Create wallet + metered contract (rate, unit, caps, scope).
2) Authorize
Issue a short‑lived allowance token (JWT) for the task.
3) Settle
Per‑call/second settlement → cryptographic receipt (SHA‑256).

5‑line integration (Python)

from mnnr import MNNR 

m = MNNR() # base_url configurable a = m.create_wallet("agent"); b = m.create_wallet("gpu") c = m.create_contract(a["id"], b["id"], 0.002) # $0.002/call m.authorize(c["id"], max_spend_usd=0.20) r = m.settle(c["id"], 1, {"job":"embed"}); print(m.verify_receipt(r))

Receipts are tamper‑evident JSON (SHA‑256 over canonical payload). Verify locally or via API.
